Sliding Window - Fixed

"The Moving Frame"

[1, 2, 3, 4, 5, 6, 7]
[---k---] Window size k
[---k---] Slide right
[---k---] Keep sliding! 
let sum = 0, maxSum = 0;

  sum += arr[i] - arr[i-k];
  maxSum = Math.max(maxSum, sum);
}
